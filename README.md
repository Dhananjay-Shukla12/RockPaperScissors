# Rock Paper Scissors

A simple console-based Rock Paper Scissors game built using Python.

The game allows a user to play against the computer in the terminal. The computer randomly chooses Rock, Paper, or Scissors and the winner is decided based on the game rules.

## Technologies Used

* Python
* Random module

## Features

* User vs Computer gameplay
* Random computer moves
* Replay option
* Simple terminal interface

## How to Run

```bash id="l2m7xq"
python main.py
```
